MBTA Live Status Homepage (unofficial)

This is the homepage for my unofficial MBTA live tracking project, built using the MBTA's V3 API.

About

This project is written in Python, using deck.gl/pydeck for the maps. It's hosted on Google App Engine, and the source code can be found on GitHub here.
